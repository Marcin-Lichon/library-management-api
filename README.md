Library Management API
A robust Spring Boot REST API for managing a library's book collection. This project focuses on clean architecture, data validation, and automated documentation.

🚀 Features
Full Book CRUD: Create, read, update, and delete books in the system.
Smart Filtering: Search for books by author using dynamic query parameters.
Data Validation: Enforced data integrity using Bean Validation (Jakarta Validation).
Global Error Handling: Clean, structured JSON error responses for all API exceptions.
Interactive Documentation: Integrated Swagger UI for real-time API testing.
Data Mapping: Clean separation between Entities and DTOs using MapStruct.

🛠 Tech Stack
Java 21 (or 17/22 depending on your version)
Spring Boot 3.x / 4.x (with Spring Framework 7)
Spring Data JPA
H2 Database (In-memory for development)
MapStruct (Object mapping)
Lombok (Boilerplate code reduction)
Springdoc OpenAPI (Swagger)
