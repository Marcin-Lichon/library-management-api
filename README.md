# Library Management API
A robust Spring Boot REST API for managing a library's book collection.

## 🚀 Features
- **Full Book CRUD**: Create, read, update, and delete books in the system.
- **Smart Filtering**: Search for books by author using dynamic query parameters.
- **Data Validation**: Enforced data integrity using Bean Validation.
- **Global Error Handling**: Clean, structured JSON error responses.

## 🛠 Tech Stack
- **Java 21** (or 17/22)
- **Spring Boot 4.x** (with Spring Framework 7)
- **Spring Data JPA**
- **H2 Database** (In-memory)
- **MapStruct** & **Lombok**
- **Springdoc OpenAPI** (Swagger)
